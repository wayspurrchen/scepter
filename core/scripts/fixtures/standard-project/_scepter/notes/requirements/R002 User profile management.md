# R002 - User profile management

Users must be able to manage their profiles including:

- Personal information (name, email, avatar)
- Preferences and settings {C002}
- Privacy controls
- Account deletion (GDPR compliance)

This integrates with {R001} for authentication and {R005} for multi-language preferences.

Implementation tracked in {T002}.
